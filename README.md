# Ipp

IPP is Ruby library for determining your public IP address using https://www.ipify.org.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ipp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ipp

## Usage

In Ruby programs:
    $ Ipp::get

In console you can use executable:
    $ ipp
